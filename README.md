# 실시간 CCTV 영상 분석을 통한 현재 날씨 정보 제공 어플리케이션
<br/>

도로교통부에서 제공하는 실시간 도로 CCTV 영상 분석을 통해 날씨를 분석하여 제공하는 어플리케이션입니다.

<br/>

### 기능

1. 장소 선택
2. 실시간 날씨 정보 및 도로 영상 제공
